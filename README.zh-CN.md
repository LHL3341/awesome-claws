## 贡献

欢迎贡献！

建议的收录标准：

- Open source or source-available
- Actively maintained
- Relevant to OpenClaw-style assistants, research tools, coding products, automation, skills, or community ecosystem
- Useful in a real-world use case, not just a demo
- Better understood as a **product** than a low-level library
- Prefer projects with a working demo, clear docs, screenshots, or a clear installation path

建议的排除标准：

- Dead repos with no maintenance signal
- Closed-source SaaS with no public code
- Generic AI lists with no relation to the ecosystem
- Low-signal clones with unclear authenticity
- Low-level infra/framework components that are not meaningful as standalone end-user products

如果可以，请添加简短描述，说明该产品能做什么。 focused on **what job the product is useful for**.
